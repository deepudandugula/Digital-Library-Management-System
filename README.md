# digital-library-management-web-application
A full-stack web application for managing and exploring books, built using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite with complete CRUD functionality and PDF access.
This project is a full-stack Book Management Web Application designed to organize and manage books related to books.The application allows users to add, view, edit, and delete books, along with ratings, descriptions, images, and mandatory PDF links. Clicking on a book title directly opens the corresponding PDF in a new tab for easy reading.

   The frontend is built using HTML, CSS, and Vanilla JavaScript for dynamic UI rendering, while the backend uses Node.js and Express.js to provide RESTful APIs. SQLite is used as the database to store and manage book records efficiently.This project follows a client–server architecture and focuses on clean CRUD operations, validation, and real-world usability.

Technologies Used :
  Frontend: HTML5, CSS3, JavaScript
  Backend: Node.js, Express.js
  Database: SQLite
  Architecture: RESTful Client–Server

⭐ Key Features :
  => Add, update, delete books
  => Mandatory PDF link for each book
  => Clickable book title opens PDF
  => Dynamic UI rendering
  => Form validation
  => EST API integration
  => SQLite database persistence
